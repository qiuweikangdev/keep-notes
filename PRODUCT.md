# Product

## Register

product

## Users

开发者、写作者、笔记爱好者。使用场景：日常记录、技术笔记、文档写作。需要一个简洁高效的 Markdown 编辑工具，支持本地文件管理和 GitHub 同步。

## Product Purpose

提供一个简约、专注、高效的 Markdown 笔记应用。核心价值：让写作回归本质，不被复杂功能干扰。成功标准：用户能快速记录想法，流畅编辑 Markdown，无缝同步到 GitHub。

## Brand Personality

简约、专注、高效

- 简约：界面干净，功能克制，视觉干扰最小化
- 专注：突出内容本身，减少认知负担
- 高效：操作直觉化，快捷键完善，多面板协作

## Anti-references

- Notion：功能堆砌导致界面复杂，学习曲线陡峭
- Obsidian：默认样式粗糙，需要大量自定义才能使用
- Word/传统编辑器：工具栏密集，界面臃肿，不符合现代审美

## Design Principles

1. **内容优先**：编辑区域最大化，UI 元素服务于内容创作
2. **渐进式复杂度**：基础功能零门槛，高级功能按需发现
3. **视觉克制**：色彩、动效、装饰元素保持最小化
4. **一致性**：多主题间保持统一的交互逻辑和信息层级
5. **原生感**：遵循桌面应用的交互习惯，快捷键、拖拽、右键菜单

## Accessibility & Inclusion

基础无障碍：键盘导航、焦点状态、基本对比度。暂无 WCAG 等级要求，后续可按需提升。
